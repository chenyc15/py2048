## Py2048 Game
Simple 2048 game written in Python.

Use W/S/A/D for moving up/down/left/right.

### Dependencies
Numpy 1.12.1

### Usage
In terminal, run

```
python main_cl.py
```

for default 4x4 board. Or run
```
python main_cl.py --width N --height M
```
for MxN board.
